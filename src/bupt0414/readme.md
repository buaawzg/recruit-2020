## TestC - Nim博弈

两个人玩游戏  
有若干堆石子, 每堆石子的数量都是有限的  
一个人选择一堆石子并拿走若干颗(不能不拿)  
如果轮到某个人时所有的石子堆都已经被拿空了, 则失败

***对于一个Nim游戏的局面 `(a1,a2,...,an)`, 当且仅当 `a1 ^ a2 ^ ... ^ an != 0`, 其中 `^` 表示异或 (xor) 运算, 先手必胜.***
