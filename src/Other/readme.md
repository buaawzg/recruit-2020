## 称重量
**百度面试**  
`100` 瓶药, 每一瓶里有 `100` 粒药  
 其中 `99` 瓶中每一粒药都是 `1g`, 剩下 `1` 瓶中的每一粒药是 `0.9g`   
 给一个秤, 只秤一次, 得到是哪一瓶里的药是 `0.9g`   
 ***药瓶编号 `1 - 100`, 第 `i` 瓶里取 `i` 粒药, 称重为 `Xg`***   
 ***`result = ((1 + 100) * 100 / 2 - X) / 0.1`***
